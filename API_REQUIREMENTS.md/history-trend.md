# 8. 大屏统计：历代色彩趋势变化与核心色值

**功能描述**：
用于首页大屏右侧“核心色彩”卡片及底部“历史趋势”面积图的渲染。该接口返回不同历史年份（从唐代 618 年至清末 1800 年）下，四种关键建筑材料（黄色琉璃瓦、青色瓦片、绿色琉璃瓦、红色墙体）的使用数量演变趋势。

* **请求地址 (URL)**: `/api/dashboard/history-trend`
* **请求方式 (Method)**: `GET`
* **请求参数 (Query)**: 无（返回全量历史时间轴数据）

### 字段说明 (Fields)

| 字段名 | 类型 | 说明 |
| --- | --- | --- |
| `years` | `Array` | X 轴时间刻度列表（对应唐、宋、元、明、清等关键年份） |
| `series` | `Array` | 趋势数据系列，对应图中的堆叠面积块 |
| `series[].materialName` | `String` | 材料/色彩名称（用于 Tooltip 和图例显示） |
| `series[].data` | `Array` | 数值数组，长度需与 `years` 一致 |
| `series[].color` | `String` | 该材料在图表中对应的渲染颜色 |
| `cultureTags` | `Array` | 用于渲染“核心色彩”下方的四个胶囊形状标签 |
| `description` | `String` | 底部的数据说明文本 |
